# Squircular Lissajous Aiming (Send Input Demo)

This is a demo of a controller aiming system that combines:

- Gyro input (gross aim)
- Displacement-based thumbstick aim (fine aim)


The goal is to make controller aim behave closer to mouse aim.

## Features
- Improved micro-adjustments  
- More stable tracking  
- Better flick consistency  

## Tested in
- Counter-Strike 2  
- Halo MCC  
- Quake Live  


## Important Notice (Read This)

This demo uses **SendInput** for mouse output.

- It may not work in games with anti-cheat systems❗ 
- It is intended for single-player games, offline testing, or unrestricted environments only❗ 


## Instructions

1. Connect a gyro-capable controller (preferably DS5 controller)
2. Run the executable  
3. Test aiming in a supported environment



For multiplayer use:
- A **hardware-based solution (Teensy USB)** is required
- This is included in the **full/premium version**
